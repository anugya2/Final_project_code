REST API Endpoints:


1. Add Protein Entry:

POST http://127.0.0.1:8000/api/protein/ - add a new record


2. Show Protein Details:

GET http://127.0.0.1:8000/api/protein/[PROTEIN ID] - return the protein sequence and all we know about it: "/api/protein/A0A016S8J7"


3. Show Domain Details:

GET http://127.0.0.1:8000/api/pfam/[PFAM ID] - return the domain and it's description: "api/pfam/PF00360/"


4. Return a list of all proteins for a given organism:

GET http://127.0.0.1:8000/api/proteins/[TAXA ID] : "api/proteins/55661/"


5. Return a list of all domains in all the proteins for a given organism.

GET http://127.0.0.1:8000/api/pfams/55661 : "api/pfams/55661/"


6. Return the domain coverage for a given protein.

GET http://127.0.0.1:8000/api/coverage/[PROTEIN ID] : "api/coverage/A0A016S8J7/"
