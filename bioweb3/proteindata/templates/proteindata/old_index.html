Testing functionalities:


1. Add User/ Create your account:

POST http://127.0.0.1:8000/api/user/ - add a new record


2. Add Tools to share:

POST http://127.0.0.1:8000/api/tool/1/ -Add tools and items to share.


3. List all Tools in the inventory:

POST http://127.0.0.1:8000/api/tool/1/ -Add tools and items to share.


4. Filtered inventory based on category:

POST http://127.0.0.1:8000/api/filter_tools/sewing -Filter inventory for sewing tools


5. Display only the tools in your account:

POST http://127.0.0.1:8000/api/filter_tools/sewing -Filter inventory for sewing tools


6. Add the tools in your account to inventory:

POST http://127.0.0.1:8000/api/filter_tools/sewing -Filter inventory for sewing tools
